Designed for security operations, event management, and public safety — delivering actionable intelligence before incidents escalate. Upload historical footage or monitor live feeds in real time.

LOCAL SETUP

Run ULTRON locally. Requires Python 3.10+ and a webcam or RTSP stream.

# Install dependencies
cd backend && pip install -r requirements.txt

# Download YOLO model
python -c "from ultralytics import YOLO; YOLO('yolo11s.pt')"

# Launch ULTRON
python main.py
# Open: http://localhost:8000

RENDER.COM (FREE)

Deploy free to Render. Auto-deploys from GitHub via render.yaml config.

# Push to GitHub
git init && git add . && git commit -m "init"
git push origin main

# On Render.com
New → Web Service → Connect repo
→ Auto-deploys via render.yaml

RASPBERRY PI (EDGE)

On-premise edge detection. Low latency, no cloud dependency required.

# config.py changes for Pi
YOLO_MODEL = "yolo11n.pt"
SKIP_FRAMES = 3

pip install -r requirements.txt
python main.py
# Access: http://PI_IP:8000
